Título: Simulador de Parquímetro

Descrição: Aplicação interativa que simula um parquímetro utilizando Programação Orientada a Objetos (POO) em JavaScript. O usuário informará um valor, e a aplicação calculará o tempo de permanência e o troco (se houver).

Tecnologias utilizadas: HTML5, CSS3, JavaScript

Pré-requisitos: Navegador atualizado (projeto foi testado e funciona no Google Chrome Versão 142.0.7444.163 64 bits - Caso utilize este navegador, recomenda-se tal versão ou superior)

Instalação:

Passos para clonar e rodar: git clone https://github.com/tayanibritto/parquimetro.git cd parquimetro abrir index.html no navegador
Como usar: Insira o valor desejado e clique em "Pagar".
Estrutura do Projeto: index.html scripts.js styles.css
